# 04 — 常见错误排查

## 上传阶段

### `no such domain`（404）

```json
{"error":"no such domain"}
```

- **原因**：`uploadUrl` 不是有效上传域名（例如用了业务 CDN 域名 `air-test-qn.airdroid.com`）。
- **处理**：改用机房对应上传域，如华南 `https://up-z2.qiniup.com`。

---

### `bad token` / `BadToken`（401）

```json
{"error":"bad token","error_code":"BadToken"}
```

常见原因：

| 原因 | 处理 |
|------|------|
| AK/SK 错误或不配对 | 核对控制台密钥 |
| token 被截断 / 复制缺字符 | 整段粘贴 uploadToken |
| 签名时去掉 Base64 的 `=` | 使用修复后的 `qiniu_prefix_uptoken.py` 重新生成 |
| bucket 与 AK 无权限 | 换有权限的密钥或 bucket |

policy 本身可解码自检：token 第三段 Base64 解码后应含 `scope`、`deadline` 等。

---

### `incorrect region`（400）

```json
{"error":"incorrect region, please use up-z2.qiniup.com, bucket is: bizdl-airdroid-com"}
```

- **原因**：上传域名与 bucket 机房不一致。
- **处理**：按提示改用 `up-z2.qiniup.com`（或控制台给出的域名），并写入 session 的 `uploadUrl`。

---

### curl 上传仍失败

- 去掉手写 `Content-Type` 与 `boundary`。
- `file` 使用 `file=@/绝对路径`。
- 确认 `key` 在 `keyPrefix` 下。

---

## 下载 / 检查阶段

### `405 Method Not Allowed`

- **原因**：对**上传域名**发 GET 下载。
- **处理**：改用**访问域名**（如 `bizdl.airdroid.com`）。

---

### `401 Unauthorized`（下载）

- **原因**：私有空间未带签名，或 `e`/`token` 错误、过期。
- **处理**：`--private` + 正确 AK/SK 重新生成 URL。

---

### `404 Not Found`（下载，已带私有签名）

签名通过但仍 404，通常是**域名与 bucket 不对应**或 **key 不存在**：

| 检查项 | 说明 |
|--------|------|
| 访问域名 | 必须绑定到上传时用的 bucket（bizdl → `bizdl.airdroid.com`） |
| key | 与上传返回的 `key` 完全一致 |
| 误用 `.cn` / 其他环境域名 | 对照控制台与 `uc.conf` |

---

### `UnicodeEncodeError`（ascii codec）

- **原因**：`--url` 含中文，旧版脚本未编码 path。
- **处理**：已修复；或改用 `--base-url` + `--key`。

---

### HEAD 上传域显示 `exists: true` 但下载失败

- **原因**：上传域 HEAD 不能代表对象在 CDN 上可读。
- **处理**：只对**访问域名 + 私有签名 URL** 做 HEAD/GET。

---

## 测试相关

### `ModuleNotFoundError: test_qiniu_prefix_uptoken`

- **原因**：在仓库根目录执行 unittest，测试文件在 `projects/qiniu-uploader/scripts/` 下。
- **处理**：`cd file && python3 -m unittest test_qiniu_prefix_uptoken.py`

### `KeyError: 'mimeLimit'`

- **原因**：`mime_limit=None` 时 policy 不含该键，却用 `policy["mimeLimit"]` 断言。
- **处理**：`assertNotIn("mimeLimit", policy)` 或传入 `mime_limit="video/*"`。

---

## 快速对照表

| HTTP | 典型含义 | 优先检查 |
|------|----------|----------|
| 401 BadToken | 上传 token 验签失败 | AK/SK、Base64 padding、token 完整性 |
| 400 incorrect region | 上传机房错 | uploadUrl |
| 404 no such domain | 上传 URL 不是上传域 | uploadUrl |
| 404（下载） | 域名或 key 不对 | base-url、bucket 绑定、key |
| 405 | 用上传域 GET | 改用访问域 |
| 579 | 回调失败 | callbackUrl 与响应 JSON |
