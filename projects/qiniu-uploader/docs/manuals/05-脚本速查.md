# 05 — 脚本速查

均在 `projects/qiniu-uploader/scripts/` 目录下执行。先 `cd projects/qiniu-uploader/scripts` 再运行下列命令。

---

## 测试

```bash
python3 -m unittest test_qiniu_prefix_uptoken.py -v
```

---

## 生成上传 session

```bash
python3 qiniu_prefix_uptoken.py \
  --access-key "$QINIU_ACCESS_KEY" \
  --secret-key "$QINIU_SECRET_KEY" \
  --bucket "bizdl-airdroid-com" \
  --key-prefix "videos/task123/device456/" \
  --upload-url "https://up-z2.qiniup.com" \
  --expires-in 1800 \
  --fsize-limit 5368709120 \
  > session.json
```

---

## 上传单文件（curl）

```bash
curl -v -X POST 'https://up-z2.qiniup.com' \
  -F "token=$(jq -r .uploadToken session.json)" \
  -F 'key=videos/task123/device456/你的文件名.pdf' \
  -F 'file=@"/path/to/file.pdf"'
```

---

## 上传本地目录

```bash
python3 upload_local_directory_to_qiniu.py \
  --local-dir "/你的目录" \
  --session-file "./session.json"
```

---

## 检查是否存在（私有）

```bash
python3 qiniu_check_object_exists.py \
  --base-url "https://bizdl.airdroid.com" \
  --key "videos/task123/device456/你的文件名.pdf" \
  --private \
  --access-key "$QINIU_ACCESS_KEY" \
  --secret-key "$QINIU_SECRET_KEY"
```

---

## 下载（私有）

```bash
python3 qiniu_download_object.py \
  --base-url "https://bizdl.airdroid.com" \
  --key "videos/task123/device456/你的文件名.pdf" \
  --private \
  --access-key "$QINIU_ACCESS_KEY" \
  --secret-key "$QINIU_SECRET_KEY" \
  --output "/tmp/out.pdf"
```

---

## 环境变量示例

```bash
export QINIU_ACCESS_KEY='你的AK'
export QINIU_SECRET_KEY='你的SK'
```

勿将密钥提交到 Git。
